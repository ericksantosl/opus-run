# OpusRun

## Sobre o projeto

OpusRun é uma aplicação web de gerenciamento de tarefas desenvolvida para fins acadêmicos e demonstração de conhecimentos em desenvolvimento front-end.

## Tecnologias utilizadas

* HTML5
* Tailwind CSS
* JavaScript

## Funcionalidades

### Landing Page

* Apresentação do sistema
* Formulário de dúvidas com validação

### Dashboard

* Visão geral das tarefas
* Opção para adicionar nova tarefa

### Página de Pendentes

* Listagem de tarefas pendentes
* Botão para concluir tarefa

### Página de Concluídas

* Listagem de tarefas concluídas
* Botão para excluir tarefa

## Como executar

1. Faça o download ou clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Navegue pelas páginas utilizando os menus disponíveis.
